# წყაროს ბარათი (Source Note) — GOST R 52726

⚠ **Processing-order correction (შენარჩუნებულია ისტორიის გამო):** batch-task-ის თავდაპირველ დავალებაში ეს ფაილი მითითებული იყო როგორც "Surge arrester / ГОСТ Р 52726" (item 10). **ეს დადასტურებულად არასწორი იყო.** ამ ვერსიაში (2.0) ჩატარებულია **სრული დოკუმენტის ვიზუალური დადასტურება** (არა მხოლოდ ბიბლიოგრაფიული მითითება), რომელმაც საბოლოოდ დაადასტურა, რომ დოკუმენტი ეხება **disconnectors-ს, earthing switches-ს და მათ საამძრავო მექანიზმებს**, და **საერთოდ არ ეხება surge arrester-ს**. მეტიც — თავად დოკუმენტის Scope (Section 1) ცალსახად **გამორიცხავს** "разъединители с ограничителями перенапряжения" (disconnectors ჩაშენებული surge limiter-ით) — ანუ დოკუმენტი თავად აშკარად განასხვავებს თავს surge-arrester-კომბინირებული ვარიანტებისგან.

## 3. Scope and Relevance for This Project

**Scope (Section 1, პარაფრაზი):** ვრცელდება ცვლადი დენის (50 ჰც) disconnector-ებზე და earthing switch-ებზე >1 კვ ძაბვაზე და მათ საამძრავო მექანიზმებზე. **გამორიცხავს:** სპეციალურ შესრულებებს (surge-limiter-კომბინირებული, plug-in ტიპის, ჩაშენებული fuse-ით, GIS-ტიპის), ასაფეთქებელ/ხანძარსაშიშ გარემოში გამოყენებას, ხშირი კომუტაციის რეჟიმს, მაღალვიბრაციულ/დარტყმით გარემოს (მაგ. ექსკავატორები).

- **10 kV outgoing feeder cubicle:** **პირდაპირ რელევანტური** — Table 2-ის (Основные параметры) სტანდარტული ძაბვის რიგში "10 (12)" კვ ფიგურირებს (სტრუქტურულად დაფიქსირებული, კონკრეტული მნიშვნელობები არ არის კოპირებული).
- **Company technical assignment/Supplier offer review:** გამოსადეგია domestic (Russian) jurisdiction-ის შემთხვევაში disconnector/earthing switch მოთხოვნების დასაბუთებისთვის, **მაგრამ NEQ სტატუსის გამო არ შეიძლება ავტომატურად ჩაითვალოს IEC 62271-102-ის ეკვივალენტად** — კონკრეტული პარამეტრული განსხვავებები ცალკე უნდა შემოწმდეს საჭიროების შემთხვევაში.
