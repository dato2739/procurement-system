# Source Note Template

Use this template to create one source note per document before any standard, regulation, or manufacturer document in `knowledge/materials_inbox/` is treated as controlled Knowledge Base evidence, per `CLAUDE.md` §3.1 and §14.1. Copy this file into `knowledge/source_cards/` under a descriptive filename (e.g. `GOST-R_55190-2012_source-note.md`) and fill in each field.

A source note does **not** replace the raw document — it is a short, citable summary written in our own words that lets agents reference the source without redistributing its protected content. See [`copyright_safe_extraction_guideline.md`](../technical_notes/copyright_safe_extraction_guideline.md) before writing one.
